# 4.wx.navigateTo和wx.redirectTo

wx.navigateTo是保留当前页面（执行的是onHide()），然后跳转到应用内的某一页面，还可以返回

wx.redirectTo是关闭当前页面（执行的是onLoad()），跳转，不可以返回

# 8.数据缓存

四类操作八种方法（每一类包括同步和异步两种方法）：

设置缓存：setStorageSync(key,value)

获取缓存：getStorageSync(key)

删除缓存：removeStorageSync(key)

清除所有缓存：clearStorageSync()

同步与异步：StorageSync，Storage

一、设置、修改缓存的方法：
同步：wx.setStoragesync('key',"Hello");
异步：wx.setStorage();
二、获取缓存的方法：
wx.getstoragesync('key');
wx.getstorage('key');
三、删除缓存
wx.removeStoragesync('key');
wx.removeStorage('key');
四、清除所有缓存
wx.clearStoragesync()不需要指定键名
wx.clearStorage()
缓存的上限最大不能超过10M

缓存不清除永久存在，缓存上限是10M

# 9.this
showModal: function (postsCollected, postCollected) {
        var that = this;
        wx.showModal({
            title: "收藏",
            content: postCollected ? "收藏该文章？" : "取消收藏该文章？",
            showCancel: "true",
            cancelText: "取消",
            cancelColor: "#333",
            confirmText: "确认",
            confirmColor: "#405f80",
            success: function (res) {
                if (res.confirm) {
                    wx.setStorageSync('posts_collected', postsCollected);
                    // 更新数据绑定变量，从而实现切换图片
                    that.setData({
                        collected: postCollected
                    })
                }
            }
        })
}

this指的是函数调用的上下文环境，如果在自定义嵌套函数中使用this指针，这样的this.setData({})写法是不对的，因为setData({})是属于Page({})结垢中某个变量的。

解决方法：在第一层自定义函数中的this还是this，可以var that = this; 然后再在内嵌的函数中用that引用this，即that.setData({)}。

this指的是当前位置的上下文，其实简单来说就是当前所在位置的最直接的结构体对象。

# 12.Page和App

在小程序中，app具有全局生命周期，而某个page的生命周期对应该页面开关

Page：页面的生命周期的控制

App：关于应用程序的方法属性

全局App变量：

var app = getApp();
var gloabal = app.globalData;

Page生命周期：

1onLoad() 页面渲染.

2.onReady() 页面就绪

3.onShow() 页面渲染

4.onHide() 页面隐藏

5.onUnLoad() 页面卸载

App只有三个生命周期函数：

1.onLaunch（启动）

2.onShow（启动后和隐藏后的消失）

3.onHide（后台隐藏）

所以，让我们考虑到音乐播放的问题：

记录页面的播放状态，但不能在页面内记录，因为页面关闭之后，所有的状态都会恢复到初始状态，不符合音乐播放状态记录的要求，所以，要有一个与页面无关的状态保持： 缓存？过了 当我页面关闭的时候，这个状态依然保持着变量，而当我的应用程序全部关闭了之后，这些所有的状态变量都会恢复到默认值 全局变量合适App({})

# 15.小程序三种路由

wx.navigateTo 保留当前页面，跳转到应用内的某个页面，但是不能跳转到tabbar页面，使用wx.navigateBack可以返回到原页面。

wx.redirectTo, 关闭当前页面，跳转到应用内的某个页面，但是不允许跳转到tabbar页面

wx.switchTab 在项目中增加了Tab栏之后，必须使用wx.switchTab()方法
